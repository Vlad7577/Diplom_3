Diplom\_3



UI автотесты для веб-приложения Stellar Burgers.



\## Используемые технологии



\- Python

\- Selenium

\- Pytest

\- Allure

\- Page Object Model



\## Что проверяется



\### Основная функциональность



\- переход в раздел "Конструктор"

\- переход в раздел "Лента заказов"

\- открытие модального окна ингредиента

\- закрытие модального окна

\- увеличение счетчика ингредиента



\### Лента заказов



\- увеличение счетчика "Выполнено за всё время"

\- увеличение счетчика "Выполнено за сегодня"

\- новый заказ появляется в разделе "В работе"
